## Topic 话题说明

| Topic | 含义 |
|-------|------|
| `/parameter_events` | ROS2 系统参数变更事件，标准话题，所有节点都有 |
| `/rosout` | ROS2 日志输出，标准话题，所有节点都有 |
| `/tf_static` | 静态坐标变换，发布相机各传感器之间的位置关系 |

### 彩色相机（Color）

| Topic | 含义 |
|-------|------|
| `/robot1/D455_1/color/camera_info` | 彩色相机内参（焦距、光心、畸变系数等标定信息） |
| `/robot1/D455_1/color/image_raw` | 彩色图像原始数据（RGB） |
| `/robot1/D455_1/color/metadata` | 彩色相机元数据（曝光时间、增益、时间戳等，JSON 格式） |

### 深度相机（Depth）

| Topic | 含义 |
|-------|------|
| `/robot1/D455_1/depth/camera_info` | 深度相机内参标定信息 |
| `/robot1/D455_1/depth/image_rect_raw` | 经过校正的深度图像原始数据（16位，单位毫米） |
| `/robot1/D455_1/depth/metadata` | 深度相机元数据（JSON 格式） |

### 外参（Extrinsics）

| Topic | 含义 |
|-------|------|
| `/robot1/D455_1/extrinsics/depth_to_color` | 深度相机到彩色相机的外参（旋转矩阵 + 平移向量），用于将深度对齐到彩色图像 |
| `/robot1/D455_1/extrinsics/depth_to_depth` | 深度相机到自身的外参（恒等变换，作为基准参考） |

## 可以额外开启的话题

当前只开启了 Depth 和 Color，如果需要更多数据，可以添加参数：

```bash
# 开启点云
ros2 launch realsense2_camera rs_launch.py \
    camera_namespace:=robot1 camera_name:=D455_1 \
    pointcloud.enable:=true

# 开启红外
ros2 launch realsense2_camera rs_launch.py \
    camera_namespace:=robot1 camera_name:=D455_1 \
    enable_infra1:=true enable_infra2:=true

# 开启 IMU（D455 内置）
ros2 launch realsense2_camera rs_launch.py \
    camera_namespace:=robot1 camera_name:=D455_1 \
    enable_gyro:=true enable_accel:=true

# 开启深度对齐到彩色
ros2 launch realsense2_camera rs_launch.py \
    camera_namespace:=robot1 camera_name:=D455_1 \
    align_depth.enable:=true
```

开启后会多出以下话题：

| 额外 Topic | 含义 |
|-----------|------|
| `/robot1/D455_1/depth/color/points` | 彩色点云（XYZRGB） |
| `/robot1/D455_1/aligned_depth_to_color/image_raw` | 对齐到彩色视角的深度图 |
| `/robot1/D455_1/infra1/image_rect_raw` | 左红外图像 |
| `/robot1/D455_1/infra2/image_rect_raw` | 右红外图像 |
| `/robot1/D455_1/gyro/sample` | 陀螺仪数据 |
| `/robot1/D455_1/accel/sample` | 加速度计数据 |
| `/robot1/D455_1/imu` | 融合后的 IMU 数据 |
